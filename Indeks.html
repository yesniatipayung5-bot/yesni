e-Tanaman

Sistem Monitoring Pertumbuhan Tanaman Pangan

Tujuan Sistem

Menyediakan sistem pemantauan pertumbuhan tanaman pangan secara digital untuk membantu mahasiswa, dosen, dan petani dalam mengelola budidaya tanaman.

Masalah yang Ingin Diselesaikan

Pencatatan pertumbuhan tanaman masih dilakukan secara manual sehingga data sering tidak lengkap, sulit dianalisis, dan memerlukan waktu lama dalam pembuatan laporan.

Cara Kerja Sistem

  • Input data tanaman melalui smartphone atau komputer.
  • Mencatat tanggal tanam, jenis tanaman, tinggi tanaman, dan kondisi tanaman.
  • Data tersimpan otomatis pada database.
  • Sistem menampilkan laporan dan grafik perkembangan tanaman.
  • Laporan dapat diunduh kapan saja.

Komponen TI yang Digunakan

Hardware

  • Smartphone
  • Laptop/Komputer
  • Sensor Kelembapan Tanah
  • Server

Software

  • Aplikasi e-Tanaman
  • Database MySQL
  • Website Monitoring

Jaringan

  • Internet/WiFi

Brainware

  • Mahasiswa
  • Dosen
  • Petani
  • Admin Sistem

Manfaat Sistem

  • Data tersimpan dengan aman.
  • Mempermudah pemantauan tanaman.
  • Mempercepat pembuatan laporan.
  • Mengurangi kesalahan pencatatan.
  • Mendukung keputusan budidaya berbasis data.

Kendala dan Solusi

Kendala: Internet tidak stabil.

Solusi: Menyediakan fitur offline.


Kendala: Pengguna belum terbiasa menggunakan sistem.

Solusi: Pelatihan penggunaan aplikasi.


Kendala: Sensor rusak atau error.

Solusi: Perawatan dan pengecekan berkala.
